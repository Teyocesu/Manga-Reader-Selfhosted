## Featured project

### Manga Reader Selfhosted

A private, self-hosted manga/manhwa reader for personal archives.

- Built with **React**, **Vite**, **Node.js**, **Express**, and **SQLite**.
- Supports `.zip`, `.cbz`, `.rar`, and `.cbr` uploads.
- Includes folder upload, chapter management, thumbnails, reading progress, webtoon/page modes, and storage quota controls.
- Designed to run privately on a local machine or small cloud VM.
- Remote private access can be done through Tailscale, without exposing the app publicly.

Repository: [Manga Reader Selfhosted](https://github.com/Teyocesu/Manga-Reader-Selfhosted)

> The project does not include manga, scraping, downloading, or external catalog integrations. It is only a reader for files the user provides.
